#Eight Lesson
## Install && Config && Deploy
### Install
  Init:
  * mkdir/cd ProjectCatalog
  * npm init
  * npm i egg --save
  * npm i egg-bin --save--dev
  * npm run dev

  Add egg-bin:
  1. Edit Port -> package.json:  "scripts": {"dev": "egg-bin dev"}
	2. Open http://localhost:7001
### Config
  Edit Port -> package.json:  "scripts": {"dev": "egg-bin dev --port PORT"}
